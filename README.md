# Cybersec Notes Portfolio

This repository contains my study notes for eJPT preparation, organized by topic and phase of assessment. The goal is to keep the material practical, searchable, and easy to review on GitHub.

## Structure

```text
cybersec/
└── eJPT/
    └── Assessment Methodologies/
        ├── information gathering/
        │   ├── information gathering summary.md
        │   ├── scoping.md
        │   ├── assessment labs.md
        │   ├── active vs passive reconnaissance.md
        │   ├── importance of information gathering.md
        │   ├── active information gathering/
        │   └── passive information gathering/
        └── footprinting and scanning/
            ├── networking primer/
            ├── host discovery/
            ├── Port scanning/
            └── Evasion and scan performance/
```

## How to Use This Repo

1. Start with the summary note for a quick overview.
2. Open the topic folders for deeper study.
3. Use the lab notes to connect theory with practice.
4. Add new notes using the same folder-based structure.

## Recommended Upload Style

- Keep topic-based folders for easy browsing.
- Use one summary file per major section.
- Keep markdown short, practical, and direct.
- Prefer diagrams, examples, and command snippets where useful.

## Image Handling

- Use the shared `cybersec/eJPT/images` folder for markdown image references.

## Notes

The folder hierarchy is kept simple so the repo stays easy to browse on GitHub. The section indexes are cleaned up to present the material as a portfolio rather than a raw dump of files.
